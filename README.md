# Jake Schmidt Portfolio

My personal portfolio website built with Vue 3 and TypeScript.

## Live Site

[jrs213.github.io](https://jrs213.github.io)

## Tech Stack

- Vue 3
- TypeScript
- UnoCSS
- Vite

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
